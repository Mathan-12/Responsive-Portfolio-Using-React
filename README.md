# Responsive Portfolio (React)

This portfolio has been converted from static HTML/CSS/JS into a React app using Vite and React Router.

## Project Structure

```text
responsive-portfolio/
  index.html
  package.json
  package-lock.json
  vite.config.js
  README.md
  public/
    assets/
      docs/
      images/
  src/
    app/
      App.jsx
      routes.jsx
    main.jsx
    pages/
      HomePage.jsx
      AboutPage.jsx
      SkillsPage.jsx
      OracleDbPage.jsx
      UiUxPage.jsx
      WebDesignPage.jsx
    styles/
      main.css
      sections.css
      shared.css
```

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Routes

- / : Home page
- /about : About Me
- /skills : Skills
- /oracle-db : Database works
- /ui-ux : UI/UX works
- /web-design : Web design works
